Earn Monero and shop Amazon using XMR Bazaar

Purse dot io was a brilliant use case for BTC. Giving users the ability to earn non-KYC BTC while offering discounts to users willing to spend BTC on Amazon items. It leveraged Amazon for shipping and delivery confirmation, instead of waiting for users to ship & confirm packages. I was saddened to see it destroyed. I haven't found a suitable replacement until XMR Bazaar.

I've written this guide to be used "as-is" with the current XMR Bazaar, but I hope these steps can be integrated into XMR Bazaar for a seamless user experience. Of course, everything is open for discussion. This guide is open-sourced on Github for anyone to submit suggestions or updates.

I hope this will motivate the developers to build "wishlist purchasing" into XMR Bazaar so users can earn non-KYC Monero and receive discounts when purchasing products with Monero.

Click Shop to learn how to order items on Amazon using XMR Bazaar & pay with Monero.

Click Earn to learn how to earn non-KYC Monero by purchasing wishlist items for other XMR Bazaar users via Amazon.

Being a retired developer, I've added notes to integrate the process into XMR Bazaar. Clicking the "dev icon" () will provide context-sensitive popups to more details, pros & cons, and mock-up suggestions how to make XMR Bazaar even better by integrating wishlist purchasing using Monero.

Thanks to Doug for taking the initiative to create XMR Bazaar, and the devs & admins for keeping everything running.
